About Jacob Wildlife Centre

Jacob Wildlife Centre is a small fictional rescue and rehabilitation centre based near Liverpool. The centre cares for British wild animals that are sick, injured or orphaned. When animals are well enough, staff release them back into safe local habitats.

The stories and animals shown in this app are inspired by real wildlife rescue work, but they have been created for educational purposes. The aim is to help visitors understand why wildlife centres are important and how people can support them.

What the centre does

  • Responds to calls about injured or orphaned wildlife.
  • Provides safe enclosures, food and medical treatment.
  • Works with local schools to teach children about wildlife.
  • Releases animals when they are ready to live in the wild again.

About this web app

This web app was created as part of a university assignment on mobile web applications. The design focuses on clear navigation, readable content and simple interactions that work well on phones, tablets and laptops.

The app demonstrates how modern web technologies can support visitor information for a small wildlife centre, without requiring a native mobile application.

Key features

  • Responsive layout that adapts to different screen sizes.
  • Animals loaded from a JSON data file using JavaScript.
